# Rental Service Website

A website for chair, table, and bathroom rental services.

## Structure

- `index.html` - Main HTML file with sections for:
  - Hero section
  - About section
  - Rentals section (chairs, tables, bathrooms)
  - Gallery section
  - Details section
  - Contact section

- `styles.css` - Stylesheet for styling
- `js/main.js` - JavaScript for interactivity
- `images/` - Folder for images and photos

## Getting Started

Open `index.html` in a web browser to view the site.
